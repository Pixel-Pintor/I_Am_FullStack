Aprende HTML5

... A pesar de las innovaciones introducidas por CSS y JavaScript en estos últimos años, la estructura creada por el código HTML sigue siendo la parte fundamental del documento. Esta estructura define el espacio dentro del documento donde el contenido estático y dinámico es posicionado y es la plataforma básica para toda aplicación. Para crear un sitio o una aplicación web, lo primero que debemos hacer es programar el código HTML que define la estructura de cada una de las páginas que lo componen...

Aprende CSS3

... Te expone a un mundo de técnicas CSS que van desde las más inteligentes hasta las alucinantes. Este libro de utilidad instantánea está repleto de ejemplos creativos y prácticas recomendadas eficaces que agudizarán sus habilidades técnicas e inspirarán su sentido del diseño. Obtendrá nuevos conocimientos sobre características familiares como flotadores y unidades, y experimentará con ideas emergentes como el diseño receptivo y las bibliotecas de patrones.

Aprende JavaScript

... JavaScript es el lenguaje de programación de la web y más desarrolladores de software lo utilizan hoy en día que cualquier otro lenguaje de programación.
Este libro es para programadores que desean aprender JavaScript y para desarrolladores web que desean llevar su comprensión y dominio al siguiente nivel. Comienza explicando el lenguaje JavaScript en sí, en detalle, de abajo hacia arriba. Luego se basa en esa base para cubrir la plataforma web y Node.js...
